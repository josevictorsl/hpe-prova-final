# HPE Prova Final

Versão pública estática e autocontida do guia de estudos de História do Pensamento Econômico.

O `index.html` embute o guia e os PDFs/slides/provas locais autorizados para esta publicação.

A pasta `audio/` contém podcasts reais em MP3, em português brasileiro, com versão reduzida e completa por tutorial, além das faixas "todos reduzido" e "todos completo". O site também inclui treino extra com relações prováveis de prova e perguntas comparativas.
